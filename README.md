# Sistema de Screening Analítico para la Orientación Vocacional (Matriz DTDC)

Módulo de diagnóstico avanzado y procesamiento estadístico multidimensional diseñado para la detección temprana de dificultades decisionales en transiciones académicas.

---

## ⚖️ Declaración Jurada, Integridad y Resguardo Deontológico
* **Uso Académico e Ilustrativo:** Los registros, matrices y algoritmos procesados en este entorno tienen una finalidad estrictamente pedagógica, de testeo dinámico y demostrativa. Las bases analizadas corresponden a simulaciones algorítmicas y no representan un mapeo poblacional efectivo vigente.
* **Anonimato Irreversible:** La estructura metodológica está diseñada bajo el principio de doble ciego, prescindiendo por completo del almacenamiento de datos filiatorios o variables cruzadas que vulneren la privacidad de los participantes.
* **Uso No Automatizado:** Las salidas de este screening ofician exclusivamente como insumo preventivo o descriptivo complementario, supeditadas al juicio clínico, la supervisión reflexiva y el criterio presencial del profesional a cargo.
